# Papikos

Papikos is a responsive kos-search interface built with React, TypeScript, Tailwind CSS, and Vite.

## Start the project

    npm install
    npm run dev

Useful commands:

    npm run build
    npm run lint
    npm run preview

## Beginner documentation

If you are learning React from this project, read [docs/BEGINNER_GUIDE.md](docs/BEGINNER_GUIDE.md). It explains the folder structure, render flow, components, props, state, hooks, TypeScript types, Tailwind classes, event handling, and the carousel.

For frontend/backend communication, read [docs/API_CALLS_GUIDE.md](docs/API_CALLS_GUIDE.md). It explains API contracts, fetch, authentication, loading states, errors, environment variables, and suggested Papikos endpoints.

For a concise description of the frontend as it exists now, read [docs/CURRENT_IMPLEMENTATION.md](docs/CURRENT_IMPLEMENTATION.md). It covers listings, media, the carousel, detail layout, pricing, payment breakdowns, responsiveness, and backend handoff points.
